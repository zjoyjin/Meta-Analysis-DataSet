## About the Dataset

This repository hosts the curated dataset and coded variables used in our meta-analysis of 67 peer-reviewed studies (2022–2025) on depression detection from social-media data.

The dataset includes:

- Study-level metadata

- Model architecture classifications (traditional ML, deep learning, hybrid)

- Label-source categories and ethical compliance coding

- Effect-size conversions (Pearson r, accuracy, F1)

- PRISMA-informed screening and coding outputs

- Because several components originate from human-subject research and contain sensitive content, access is controlled and requires a formal request.
